Introduction:
This project is a simple software of a order entry system for a company that sells bicycles. In this software
as a system user , we will have a facilities to add a customer , add/remove a inventory , add a stock for that inventory , 
place a order for a customer , display order for a customer based on a date , display the inventory and display the customer list.
While adding a stock for a inventory , the particular item should be present in a inventory , otherwise that item cannot be added in a stock.Similarly , if an item doesnot exist or if doesnot have
enough quantaties in a stock then that item is not put in order.

Prerequisities:
To run this project on different IDE like eclipse or eclipse , we should first install the MAVEN.

CItemController
This is the main controller for contoling all the other controllers.

methods : 
provideCItemAccess()
This method allow the user to enter the choice when the program first starts. When the user enter a choice this 
method call executeChoice() method to execute the user input.

executeChoice(int choice)
This method execute the user input. Based on user input this method will call the different controller classes and their functions. like
CustomerController , InventoryController and StockController.

CustomerController
This controller controls all the action perform on customers.

methods:

executeCustomer(int)
This method is called from CItemController class when the user input a choice that delas with customer. This method will give system user a option 
weather to select : add a new customer or place order for a customer.

ExistingCustomer()
This method will check if the entered id match with a existing customer or not.

ExecuteOrder(Customer)
This method will execute the customer order when the order is placed.

displayOrder(int)
This method will display the customer order.

InventoryController
This controller class controls all the action perform on Inventory

methods:

executeInvntory(int)
This method is called from CItemController class when the user input a choice that deals with inventory.
This method is basically used for adding a item in inventory , removing a item from inventory and listing a inventory.

displayInventory()
This method display the inventory.

StockController
This controller class controls all the action perform on Stock

methods:
executeStock(int)
This method is used for adding a stock for a particular item from a inventory. If the item is not found then the stock is not added.
